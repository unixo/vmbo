Strutture dati

Queste sono le strutture dati con una loro breve descrizione:
active_pageStruttura per rappresentare una pagina associata ad un frame
active_pages
frameStruttura per la rappresentazione di un frame di memoria
free_frames
io_dev_dataStruttura per la configurazione del dispositivo di I/O. Il campo "req_count" viene incrementato quando una nuova richiesta di I/O viene inserita nella coda. Il tempo entro il quale viene servita la richiesta e' limitato inferiormente da Tmin e superiormente da Tmax
io_entryStruttura per la rappresentazione di una richiesta d'I/O
io_requestsLista FIFO di richieste di I/O
mmu_dataParametri di configurazione del modulo MMU
mmu_shared_dataStruttura per contenere la richiesta attuale alla MMU
optionParametri da riga di comando (versione lunga)
pageStruttura per la rappresentazione di una pagina virtuale
procStruttura per la rappresentazione in memoria di un processo
proc::proc_stats
used_frames
